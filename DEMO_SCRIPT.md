# PitchX TrustOps Demo Script

## 3-Minute Flow

1. Open Existing Company mode and click `BrowserWire`.
2. Start analysis and call out: "Before any agent speaks, PitchX ranks evidence with NVIDIA TrustOps."
3. Point to the research message: "NVIDIA NeMo Retriever ranked evidence passages."
4. In the boardroom, open the status panel and point to `Agent TrustOps`.
5. Say: "Verified claims must map to reranked evidence. Unsupported claims are downgraded to assumptions."
6. Let the Devil's Advocate round finish, then show the board resolution and dual gauges.
7. Click HR Panel, add a candidate with an email/phone in the resume text, and run evaluation.
8. Point to `NVIDIA HR Privacy Guard`: "PII is detected and redacted before HR evaluation."
9. Click `Approve & Execute Actions`.
10. Point to the execution log: "NemoGuard checks every proposed email or job post before the action is allowed."

## One-Liner

PitchX uses NVIDIA NIM not just to generate text, but to rank evidence, guard agent behavior, redact sensitive hiring data, and approve real-world actions.

## Fallback Line

If live APIs are slow, PitchX uses cached demo evidence and labels it clearly; the TrustOps flow still runs with deterministic fallback scoring so the agent does not break off the golden path.

## Judge Q&A

- Trust: "Every verified claim is tied to ranked evidence or downgraded to an assumption."
- Memory: "Agents store scoped memories; out-of-scope writes are rejected and logged."
- NVIDIA: "NIM powers generation, evidence ranking, safety guard checks, and the action firewall."
- Impact: "This is the board meeting and operating review a founder can run before spending six months building the wrong thing."
